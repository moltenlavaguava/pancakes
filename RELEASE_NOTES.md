# Welcome to pancakes! 🥞

### How to run:
- *Windows*: Select `pancakes.exe` below and run the file downloaded. If you get a window saying something similar to "Windows Protected your PC," click "More info" and select "Run anyway." If you're worried about this project being a virus, feel free to put it in a virus checker ;)
- *macOS*: Select `pancakes-macOS-Universal.zip` and unzip the file downloaded. Then, run the app within. You will likely need to allow the app to run for the first time in System Preferences.
- *Linux*: Download `pancakes-linux` and run it. You may need to give yourself permission to run it. If you're on linux though, I'm sure you can figure it out :D.
