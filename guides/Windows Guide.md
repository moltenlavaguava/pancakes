## Installing Pancakes

1. First, download pancakes from [https://www.riptyde.net/pancakes](https://www.riptyde.net/pancakes). *Alternatively, you can get pancakes directly from the github repository: [https://github.com/moltenlavaguava/pancakes/releases](https://github.com/moltenlavaguava/pancakes/releases). Just make sure to select the correct download (pancakes.exe)!*
3. Unzip `pancakes.zip`. Typically you can find this in the top right of your browser or in File Explorer. 
4. Run (double click) `pancakes.exe`.
5. You will likely get a popup saying "Windows protected your PC." To bypass this, click `More info` at the top of the menu and navigate to `Run anyway` (see below).   

![Windows Protect PC](windows-protect-pc)

![Windows Protect PC 2](windows-protect-pc-2)

## Installing Python + Dependencies

1. Click `Install Python` at the top of the menu, and follow the instructions. You will likely need to install the package `uv`.
2. Once you finish, make sure to close and reopen `pancakes.exe`.

## Installing Virtual Environment + First Project

**Important**: If python is not detected after installing *(and restarting!)* python using pancakes, restart your computer and reopen `pancakes.exe`.

1. Select `Setup Virtual Environment`.
3. Click `Select` to select the folder you want your project to be in. If you're unsure, `Documents` or `Desktop` is a good choice.
5. Enter a project name into the text box which says `Project name…`
6. Click `Create`
7. Click `Close`
8. Your project has now been made! To work in it, use VS Code or your preferred text editor and open the project folder pancakes created. Example: if you created a project `MyProject` within `Documents`, then open the `MyProject` folder in VS Code.
