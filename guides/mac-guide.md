## Installing Pancakes for macOS

Pancakes is not a verified program from Apple, so a few steps must be followed to allow Mac users to run Pancakes for the first time.

1. Download Pancakes for macOS from <https://riptyde.net/pancakes/>
   - Alternatively, you can get Pancakes directly from the GitHub repository:  
     <https://github.com/moltenlavaguava/pancakes/releases>  
     Make sure to select the correct download: `pancakes-macOS-universal.zip`

2. Unzip `pancakes-macOS-Universal.zip` and run `pancakes.app`.

3. An error will pop up saying Pancakes cannot be opened because it is not from a verified developer.

![Pancakes cannot be opened because the developer cannot be verified](page-1-image-1)

4. Click **Cancel**.
5. Open **System Preferences** and click **Security & Privacy**.

![System Preferences with Security & Privacy highlighted](page-2-image-1)

6. Check to make sure the program you are approving to run is Pancakes. Then click **Open Anyway**.

![Security & Privacy with Open Anyway highlighted](page-2-image-2)

7. Click **Open** to run Pancakes.

![macOS security prompt with Open highlighted](page-3-image-1)

8. After doing this for the first time, you should not need to do it again. After that, you can simply run the Pancakes app.

To continue, follow the guides within Pancakes itself. Good luck!


## Installing Python + Dependencies

1. Click `Install Python` at the top of the menu, and follow the instructions. You will likely need to install the package `uv`.
2. Once you finish, make sure to close and the pancakes app.

## Installing Virtual Environment + First Project

**Important**: If python is not detected after installing *(and restarting!)* python using pancakes, restart your computer and reopen the pancakes app.

1. Select `Setup Virtual Environment`.
3. Click `Select` to select the folder you want your project to be in. If you're unsure, `Documents` or `Desktop` is a good choice.
5. Enter a project name into the text box which says `Project name…`
6. Click `Create`
7. Click `Close`
8. Your project has now been made! To work in it, use VS Code or your preferred text editor and open the project folder pancakes created. Example: if you created a project `MyProject` within `Documents`, then open the `MyProject` folder in VS Code.
